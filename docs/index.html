Markdown to Word Converter

一个功能强大的Markdown到Word文档转换器，支持表格、Mermaid图表、中文格式等特性。

✨ 主要特性

📝 完整的Markdown支持 - 标题、列表、代码块、引用、链接等
📊 表格转换 - 支持复杂表格，保留格式和样式
📈 Mermaid图表 - 自动转换流程图、序列图、甘特图等为图片
🎯 智能清理 - 自动处理重复标题和格式问题
🌏 中文优化 - 完美支持中文内容和编号系统

🚀 快速开始

安装

git clone https://github.com/Aikrce/autarl.git
cd autarl
pip install -r requirements.txt

命令行使用

# 基本转换
python markdown_converter.py input.md

# 指定输出文件
python markdown_converter.py input.md -o output.docx

# 禁用Mermaid支持
python markdown_converter.py input.md --no-mermaid

Web界面

# 启动Web服务
python web_app.py

# 在浏览器中访问
http://localhost:8080
